# Election-Analysis-PowerBI
Developed a dynamic Power BI dashboard analyzing India’s 2024 General Election results, delivering insights from national to constituency level. Applied advanced DAX (MAXX, FILTER) for winner logic, built robust data models, and designed an interactive, drill-down enabled UI to uncover alliance performance, vote share trends, and regional dominance
